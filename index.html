About the Project

Built out of curiosity

Growing up, we played the games — but we rarely asked where they came from. Who built them? What was happening in the world when Doom shipped, or when Nintendo nearly went bankrupt? Those questions had no good answer that felt designed, searchable, and worth reading.

Level History started as an attempt to fix that. A platform where the history of video games is presented with the same care as the games themselves — clean, structured, and genuinely interesting to explore.

This project was built as a Portfolio Project for Holberton School . The full source code is available on GitHub.
